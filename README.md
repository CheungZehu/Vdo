# 微豆

> A Vue.js project

技术栈 vue + vue-router2 + vuex + axios + Mese-ui

## 快速使用

``` bash
# 克隆项目到本地
git clone https://github.com/CheungZehu/Vdo.git

# 安装依赖
npm install

# 在 localhost:8080 启动项目
npm run dev

```
